# Dify PPT Generator

An internal tool that generates brand-compliant PowerPoint slides through a simple chat interface.

## What It Does

Describe the slide you want in plain English, and the tool generates a Dify-branded PPT with the correct fonts, colors, layout, shadows, and logo — ready to use.

## Supported Inputs

- **Text** — describe what you want on the slide
- **Image** — upload or paste a screenshot of an existing slide to redesign it
- **File** — upload .pptx, .pdf, .txt, or .md to extract content and turn it into a slide

## Supported Layouts

- **3-column** — 3 cards in a single row
- **3+2** — 3 cards on top, 2 on bottom (for 5 items)
- **4-column** — 4 cards in a single row

## How to Use

1. Open the link
2. Type your slide content or upload a file/image
3. Download the generated PPT
4. Want changes? Just say "change the title to..." or "add a new card" — the tool remembers context

## Brand Compliance

All slides are automatically generated with:
- Söhne / Inter fonts
- Dify blue (#1032F5) accent color
- Correct shadows, card styling, and spacing
- Dify logo in the bottom-right corner
- Page number in the bottom-left corner

No manual styling needed.
